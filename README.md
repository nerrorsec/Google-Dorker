# nDorker
Automate Dorking Faster and Better than Ever.

## What can it do?
1. Google Dorking
2. Vendor Dorking
3. Github Dorking
4. Shodan Dorking

## Get nDorker
```bash
git clone https://github.com/nerrorsec/nDorker
cd nDorker
```

## Usage
```bash
python3 nDorker.py -d example.com
```
